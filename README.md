# Team Project

## Project Overview
This repository contains a **university group coursework project** completed as part of a team project module.

In a group of **five**, we designed and implemented a management system for a **fictional restaurant called *Lancaster’s***. The system was created to simulate a real-world restaurant management solution, with a strong focus on software design, documentation, and system handover.

For full context on the project, design decisions, and implementation details, please refer to the **Handover Report** located in the **ProjcetBinderFinal** folder

## Project Context
- **Coursework type:** University Team Project  
- **Team size:** 5  
- **Client:** Fictional restaurant – *Lancaster’s*  
- **Team name:** Code Crafters  
- **Team number:** 30  

This project was developed to reflect professional software development practices, including full documentation suitable for future developers or stakeholders.

## Technologies Used
- **Programming Language:** Java  
- **Application Type:** Desktop application  
- **Packaging:** JAR file  
- **Target Platform:** Windows  
- **Version Control:** Git & GitHub
